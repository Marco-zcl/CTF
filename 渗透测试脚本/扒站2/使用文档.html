软件介绍

小飞兔下载(www.xftsoft.com)是一款只需输入一个网址就能下载一个网站的软件，它可以从Internet下载你想要的网页，主要是用来快速搭建网站、深层分析网站、网站克隆、离线浏览等。

模块：整站下载、单页下载、图片下载
特点：转换网站编码、自定义特殊标签、智能识别链接、内置浏览器可视化操作、简繁互转、英文网页翻译、替换文本、删除代码等

运行环境

windows server 2008 R2及以上版本 | windows 7以上版本，windows系统需安装环境Microsoft .NET Framework 4.6以上

内置浏览器

作用：获取Cookie和UserAgent。
1、自动安装：打开软件，点击菜单栏的【帮助】，在下拉菜单选择【安装插件】，在打开窗口选择安装【WebView2】
2、手动安装：如果无法打开浏览器，需要下载安装：Microsoft Edge WebView2
2.1、找到【常青版独立安装程序】
2.2、32位系统下载【x86】安装，64位系统下载【x64】安装

报错一：应用程序无法正常启动(0xc000007b)

修复工具：DirectX修复工具
下载地址：https://dl.pconline.com.cn/download/360074-1.html

报错二：运行环境.Net Framewordk没有安装

安装环境：Microsoft .NET Framework 4.6.2 及以上版本
.NET4.6.2下载地址：https://download.visualstudio.microsoft.com/download/pr/8e396c75-4d0d-41d3-aea8-848babc2736a/80b431456d8866ebe053eb8b81a168b3/ndp462-kb3151800-x86-x64-allos-enu.exe
.NET4.8.1下载地址：https://download.microsoft.com/download/4/b/2/cd00d4ed-ebdd-49ee-8a33-eabc3d1030e3/NDP481-x86-x64-AllOS-ENU.exe

检查运行环境是否安装成功

点击[开始]，在搜索框输入 cmd ，在打开窗口中输入 reg query "HKLM\Software\Microsoft\NET Framework Setup\NDP" /s /v version | findstr /i version | sort /+26 /r
